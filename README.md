# ミニカンファレンス用サンプルリポジトリ

## 概要

このリポジトリは、2026/05/09 開催の技育プロジェクトミニカンファレンスにて実施した、Web ページ制作ハンズオン用のリポジトリです。

イベント詳細：https://geek.supporterz.jp/extra/miniconference/202605

## 開発環境のセットアップ

Node.js v22 以上が必要です。

### パッケージインストール

```bash
npm install
```

### 起動

```bash
npm run dev
```

起動後、http://localhost:5173 にアクセスしてください。
